## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- MySQL 8.0+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/rakeshgowdabp05/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env
# Edit .env with your DB credentials

# Start the server
node server.js
```

The app will be live at:
```
http://localhost:3000
```
